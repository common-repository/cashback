== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. The plugin connects to our API and installs the needed tables, then scans your posts for words.
3. Use the Cashback for Wordpress screen to register/login
4. Read the How it works page

== Frequently Asked Questions ==

Q: Do i need a 24/7 Discount account?
A: Yes, you can register a new account or login into an existing one.
Q: Where do the words link to?
A: They link to the retailer pages on http://www.247discount.nl
Q: How are the links made?
A: They are scanned and adjusted after our API imports a series of keywords. These are imported from http://www.247discount.nl
